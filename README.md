# pg-copy

```
USAGE:
    pg-copy.exe --origin-database-url <ORIGIN_DATABASE_URL> --target-database-url <TARGET_DATABASE_URL> --origin-table-name <ORIGIN_TABLE_NAME> --target-table-name <TARGET_TABLE_NAME>

OPTIONS:
        --origin-database-url <ORIGIN_DATABASE_URL>    
        --origin-table-name <ORIGIN_TABLE_NAME>
        --target-database-url <TARGET_DATABASE_URL>
        --target-table-name <TARGET_TABLE_NAME>

```
